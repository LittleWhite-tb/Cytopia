Parameters

angleDirection in relation to forward from 0 to 360 degrees. Larger angles will be reduced to this range using angles % 360. 0 = directly in front. 90 = directly to the right. 180 = directly behind. 270 = directly to the left.
distanceThe distance from the listener, from 0(near/loud) to 255(far/quiet).
loopsHow many loops should be played. Default: 0 (play one time)

Play loaded sound effect in 3d sound. Assume listener is facing forward which is perpendicular to up direction of game world. Up game world direction = +y in openal, isometric xy = openal xz plane.
